## üí° Recommendations

### For Product Images

1. **Source Options:**
   - Use Unsplash/Pexels (free, commercial-use)
   - AI generation (Midjourney, DALL-E)
   - Stock photography (Shutterstock)

2. **Image Requirements:**
   - Primary: 1200x1200px (square)
   - Gallery: 1200x800px (landscape)
   - Thumbnail: 400x400px (auto-generated)
   - Format: WebP for optimization

3. **Upload Strategy:**
   - Batch upload via Media API
   - Link to products via `images` field
   - Test CDN delivery

### For Collections

1. **Collection Strategy:**
   - **New Arrivals**: Auto-populate with `is_new=true`
   - **Bestsellers**: Manual curation of top 10
   - **Summer Sale**: Link all `on_sale=true` products
   - **Eco-Friendly**: All Eco Fashion brand items
   - **Premium**: Featured items >$200

2. **Collection Schema:**
   ```json
   {
     "name": "New Arrivals",
     "slug": "new-arrivals",
     "description": "Check out our latest products",
     "product_ids": [1, 3, 8, 15, ...],
     "is_featured": true,
     "display_order": 1
   }
   ```

### For Reviews
